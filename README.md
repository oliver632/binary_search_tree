# binary_search_tree

This is a library, that can make a binary search tree from any array of comparable elements. 

I've added the following methods to the Tree class
- #build_tree(array, start_pos, end_pos)
- #insert(data)
- #delete(data)
- #level_order
- #find(data)
- #preorder(&block)
- #inorder(&block)
- #postorder(&block)
- #height(node)
- #depth(node)
- #balanced?
- #rebalance
